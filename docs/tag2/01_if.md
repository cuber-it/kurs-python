# `if` / `elif` / `else`

## Worum geht's

Mit `if` reagiert dein Programm auf Bedingungen. Pflicht: **Einrückung** des Blocks (Standard: 4 Leerzeichen).

## Grundform

```python
if alter < 18:
    print("Minderjährig")
elif alter < 65:
    print("Erwachsen")
else:
    print("Im Ruhestand?")
```

- `if` — der Anfang
- `elif` — beliebig viele weitere Bedingungen (kurz für „else if")
- `else` — fängt den Rest, optional

## Einrückung

- **Pflicht** — Python hat keine Klammern wie C/Java.
- **Konsistent** — Spaces! keine Tabs!
- **Konvention** — 4 Leerzeichen, 2 sind auch ok, 1 zu wenig, schlecht lesbar.

```python
if x > 0:
    print("positiv")        # eingerückt = im if-Block
print("immer")              # nicht eingerückt = nach if
```

## Wahrheitswerte / Truthiness

Diese Werte sind **falsy** (gelten als `False`):

- `False`
- `None`
- `0`, `0.0`
- `""` (leerer String)
- später auch: `[]`, `{}`, `()` (leere Sammlungen)

**Alles andere ist truthy.**

```python
text = input("Sag was: ")
if text:                    # leerer String -> False
    print(f"Eingabe: {text}")
else:
    print("nichts.")
```

## Bedingungen kombinieren

```python
if x > 0 and x < 10:
    ...
if x < 0 or x > 100:
    ...
if not vorhanden:
    ...
```

**Verkettete Vergleiche** sind oft schöner:

```python
if 0 < x < 10:
    ...
```

## Einzeiler: ternärer Ausdruck

Aufbau: `wert = a if bedingung else b`

```python
n = 5
art = "gerade" if n % 2 == 0 else "ungerade"
```

# Macht dasselbe
wert = None
if bedingung:
    wert = a
else:
    wert = b

# Das prinzipiell auch
wert = b
if bedingung:
    wert = a
    
Praktisch für **kurze Zuweisungen**. Bei komplizierter Logik lieber ein normales `if`.

## Stolpersteine

- **`=` statt `==`** in der Bedingung → SyntaxError.
- **Doppelpunkt vergessen** → `SyntaxError`.
- **Einrückung gemischt (Tabs + Spaces)** → `IndentationError`.
- **`if x = 5:`** geht nicht — Zuweisung ist eine Anweisung, kein Ausdruck. (Außer mit Walross `:=`!)
- **Vergleich mit `None`** → `if x is None`, nicht `if x == None`.

## Schnellreferenz

```python
if  bedingung:
    ...
elif andere_bedingung:
    ...
else:
    ...

# kombinieren
and  or  not

# verketten
0 < x < 10

# Einzeiler
wert = a if cond else b
```
