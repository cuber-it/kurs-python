# Python-Kurs

Kursunterlagen, Übungen und Beispiele für einen fünftägigen Python-Kurs.

## Struktur

- `tag1/` – `tag5/` – Material und Übungen für die jeweiligen Kurstage
- `materialien/` – Übergreifende Unterlagen (Slides, Cheatsheets, Handouts)
- `extras/` – Vertiefende Themen, Bonusaufgaben, weiterführende Beispiele

## Voraussetzungen

- Python 3.10 oder neuer
- Ein Editor deiner Wahl (z. B. VS Code, PyCharm)
- Optional: `git` zum Mitziehen von Updates

## Einrichtung

```bash
git clone <repo-url> kurs-python
cd kurs-python
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
```

Falls einzelne Tage zusätzliche Pakete benötigen, liegt im jeweiligen Tagesordner eine `requirements.txt`.

## Arbeitsweise

Pro Kurstag gibt es einen eigenen Ordner mit Beispielen und Übungen. Eigene Lösungen am besten in einem Unterordner `loesungen/` ablegen, damit der Originalstand erhalten bleibt.

## Lizenz

Nutzung im Rahmen des Kurses. Weitergabe nur nach Absprache.
